Subunternehmer Drittland Kundendaten Zugriff

Der Auftragsverarbeiter wird die Kundendaten ausschließlich auf Servern innerhalb der Europäischen Union oder des Europäischen Wirtschaftsraums verarbeiten.

Abweichend hiervon behält sich der Auftragsverarbeiter das Recht vor, Daten zur Systemwartung in ein Drittland (insb. USA) zu übermitteln, sofern ein Subunternehmer dort ansässig ist. Ein direkter Zugriff auf unverschlüsselte Daten ist dabei nicht ausgeschlossen.

Eine Beauftragung weiterer Unterauftragsverarbeiter bedarf der vorherigen schriftlichen Zustimmung Information des Verantwortlichen.
